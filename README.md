# Whatsapp-like End to End messaging system

## Command list

- `signup <username> <password>`
- `login <username> <password>`
- `join <groupname>`
- `create <groupname>`
- `senduser <username> <text/file> <text block/file data>`
- `sendgroup <groupname> <text/file> <text block/file data>`
- `list`
- `quit`
